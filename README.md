# LLM Experiments

Kumpulan eksperimen praktis seputar Large Language Models (LLM), mencakup penggunaan pipeline HuggingFace, fine-tuning, evaluasi model, dan membangun arsitektur transformer dari scratch.

## Notebooks

1. [LLM dengan Python](https://colab.research.google.com/drive/1mBQP66i5bVVB3IjFeFRRrITrMeXcCKcv) - pipeline HuggingFace untuk summarization, text generation, translation, dan QA; fine-tuning dengan Trainer; evaluasi dengan BLEU, ROUGE, METEOR, Exact Match, Toxicity.
2. [Model Transformer PyTorch](https://colab.research.google.com/drive/1hH7WQBmzcUuuhW7K4s34juoBnmgJfQmo) - membangun arsitektur transformer dari scratch: Input Embeddings, Positional Encoding, Multi-Head Attention, Encoder, Decoder, dan Full Transformer.
